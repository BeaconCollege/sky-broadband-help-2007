Step 2

To check the status of your wireless connection with the Sky wireless router:

  1. From your desktop, click on START and select CONTROL PANEL

     

  2. From the Control Panel window, select Network and Internet Connections

  3. Select Network Connections

  4. In the Network Connections window, look for the Wireless Connection icon

  5. If the Wireless Connection icon has a yellow triangle with an exclamation mark (!) where the red cross is shown in the screen above, you will need to follow Steps 6 – 18 below to manually configure your IP settings

    • If the Wireless Connection icon does not have a red cross or yellow triangle, this means you are successfully connected to the Sky wireless router - if you are still experiencing wireless connection problems, go to check your web browser’s proxy settings
     

  6. To manually configure your IP Settings:

    • In the Network Connections window, Right Click on the Wireless Connection icon, then click on Properties at the bottom of the menu

  7. In the Wireless Network Connection Properties window, scroll down in the ‘This connection uses the following items’ window until you see Internet Protocol (TCP/IP) – click this once, then select Properties

  8. In the Internet Protocol (TCP/IP) Properties window:

     

  9. Select Use the following IP address

     

  10. Set the IP address to: 192.168.0.X (where X must be a number of your choice between 2 and 254 and should be unique for each PC connecting to the Sky wireless router)

     

  11. Set the Subnet Mask to 255.255.255.0

     

  12. Set the Default gateway as 192.168.0.1

     

  13. Select Use the following DNS server addresses

     

  14. Set the Preferred DNS server to 192.168.0.1

     

  15. Leave the Alternate DNS server blank

     

  16. Click OK

  17. Click OK again on the Wireless Network Connection Properties window

     

  18. Restart your computer and open a browser window to see if this has resolved the issue
