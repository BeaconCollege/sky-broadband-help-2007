Enable the Windows Wireless Zero configuration service

Enabling the Windows Wireless Zero configuration service.

  1. From your desktop, go to Start and select Control Panel

     

  2. In the Control Panel window, select Performance And Maintenance

     

  3. In the Performance and Maintenance window, select Administrative Tools

  4. In the Administrative Tools window, double click on Services

  5. In the Services window, scroll down until you see Wireless Zero Configuration – double click this

  6. If the Startup type if set to Disabled, click on the drop down menu and select Automatic, then click on Apply

  7. In the same window, you should now be able to click on the Start button under Service status – do this now

     

  8. The following screen will pop up briefly as the service is starting

  9. Wait until the Service Control screen closes itself (do not select Close), then select OK in the Wireless Zero Configuration Properties window

     

  10. Now go back and follow the steps for enabling Windows to manage your wireless connection
