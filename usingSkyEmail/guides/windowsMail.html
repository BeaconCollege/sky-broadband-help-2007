Windows Mail

  1. Launch ‘Windows Mail’ from the start button

  2. Click Tools then select Accounts

  3. Click Add…

  4. Select E-mail Account, click Next

  5. Type in your “Display Name” (will appear as the ‘from’ name on emails you send) then click [Next]

  6. Enter your email address (e.g.: john.smith@sky.com) in lowercase letters, with no spaces, and then click Next

  7. Leave ‘Incoming e-mail server type” drop down box as POP3. In ‘Incoming mail (POP3 or IMAP) server’ enter: pop.sky.com (all lowercase, no spaces). In ‘Outgoing e-mail server (SMTP) name’ enter: smtp.sky.com (all lowercase, no spaces). Leave “Outgoing server requires authentication” Unchecked, click Next

  8. In the ‘E-mail username’ box enter the first part of your @sky.com email address (everything before the @ sign). In the Password box, enter the password for your @sky.com email account.

    Tick the “Remember password” box if you would like Microsoft Mail to remember your password so that you do not need to enter it each time you check for any new email.

    Click Next

  9. Leave the “Do not download my e-mail at this time” box unchecked, and click Finish
